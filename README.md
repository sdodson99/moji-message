# :rocket: Moji Message

An epic web app for encoding a message with emojis.

Website: https://emoji.seandodson.com/

## Table of Contents

- [Tech Stack](#tech-stack)
- [How to Run Locally](#how-to-run-locally)
- [Contributing](#contributing)

## Tech Stack

- NextJS
- TypeScript
- Tailwind
- Google Analytics
- Firebase Hosting

## How to Run Locally

1. Install packages.

```
pnpm install
```

2. Start the application.

```
pnpm dev
```
